# 📝 To-Do List Application (Python CLI)

A simple Command Line To-Do List Application built using Python. It allows users to add, view, and delete tasks easily.

## 🚀 Features

- View all tasks
- Add new tasks
- Delete tasks
- Simple menu-driven interface
- Handles invalid input

## 🛠️ Technologies Used

- Python
- Lists
- Functions
- Loops
- Exception Handling

## 📂 Project Structure

todo-list/
│
├── todo.py
└── README.md

## ▶️ How to Run

1. Install Python
2. Clone the repository:
   git clone https://github.com/your-username/todo-list.git

3. Go to project folder:
   cd todo-list

4. Run the program:
   python todo.py

## 💻 Usage

After running, you will see:

To-Do List Application
1. View Task
2. Add Task
3. Delete Task
4. Exit

Enter:
1 → View tasks  
2 → Add task  
3 → Delete task  
4 → Exit
